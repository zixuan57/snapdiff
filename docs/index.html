snapdiff

Zero-config CLI for visual regression testing. One command to capture, one command to compare.

View on GitHub

Quick Start

npx snapdiff-cli init
npx snapdiff-cli diff
npx snapdiff-cli status

Features

Zero Config

Run snapdiff init and it sets everything up automatically.

Pixel-Level Diff

Powered by pixelmatch, accurate to the individual pixel.

HTML Report

Side-by-side comparison with baseline, current, and diff overlay.

Parallel Capture

Captures multiple pages in parallel with 3 concurrent workers.

CI Ready

Works in any CI pipeline. GitHub Action included.

Mask Regions

Exclude dynamic areas like ads or animations from comparison.
